About DFSG Review

The DFSG Review project aims to improve the quality of the Debian package repository by providing assurance that every package adheres to Debian’s strict interpretation of Debian's Free Software Guidelines (DFSG), which is further defined within Debian's Policy Manual (Policy). Using Ubuntu’s Code of Conduct as a guide, and historical frustrations as fuel, we aim to get to the heart of quality control problems and build permanent solutions.

DFSG Checker (DCheck) helps Debian Developers review packages for compliance with DFSG and Policy.
